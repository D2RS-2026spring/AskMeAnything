## 💬 Ask Me Anything

> "Ask Me Anything"（AMA）—— 有任何问题，尽管问！

**AMA** 是一种源自互联网社区的开放式问答形式，核心就是"**问我任何问题**"——不限主题，不设门槛。

在本课程中，我们开设 AMA 渠道，欢迎同学们就以下内容提问：

- 📚 课程内容、作业、考核
- 🔧 工具使用、代码报错、环境配置
- 🧠 科研方法、可重复性研究
- 🤔 任何你好奇但不知道问谁的问题

### 如何提问？

👉 [点击这里提交你的问题（GitHub Issue）](https://github.com/D2RS-2026spring/AskMeAnything/issues/new?template=ama.md&title=[AMA])

提问时请注意：
- 问题越具体，回答越有帮助
- 欢迎提问"听起来很蠢"的问题，没有蠢问题
- 所有问题和回答都是公开的，对其他同学也是宝贵的资源
